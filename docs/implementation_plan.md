# サイト開発計画：たまコーヒーハウス

たまごっちパラダイスの攻略・交流サイト「たまコーヒーハウス」を作成します。企画書に基づき、Y2K風の可愛いデザインを採用し、遺伝シミュレーターと進化条件リストを最優先で実装します。

## ターゲットとビジョン
*   **ターゲット**: たまごっちをライトに楽しむ若者、既存Wikiの難民、英語UIが苦手なユーザー、海外の一部ユーザー。
*   **特徴 (USP)**: 直感的なUI（可愛いイラストで選べるシミュレーター）、日/英対応、スマホファースト、Y2Kデザイン。

## 技術構成 (Tech Stack)
*   **フレームワーク**: `Next.js` (App Router) - 高いSEO性能、高速なページローディングを実現します。初期の広告なしSEO強化に最適です。
*   **スタイリング**: `Vanilla CSS` - 柔軟でリッチなY2Kデザイン、特有のアニメーションを実現します。
*   **ホスティング**: `Vercel` (推奨) - 無料枠で開始でき、後の広告（AdSense）導入やアクセス増加時のスケーリングも容易です。
*   **(Phase 2) データベース**: `Firebase` または `Supabase` (掲示板機能用。優先度に応じて後で追加します)
